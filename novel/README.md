# Job Fit Agent 项目知识小说

这是一组用小说方式讲解本项目的学习笔记。它不是严格的技术文档，而是把你这次做项目时遇到的真实问题串成故事：简历读取、JD 匹配、Web 页面、AI 分析、报告导出、项目整理和面试表达。

建议阅读顺序：

1. `chapter_01_项目醒来.md`
2. `chapter_02_简历进门.md`
3. `chapter_03_技能树法庭.md`
4. `chapter_04_网页有了呼吸.md`
5. `chapter_05_AI只在被邀请时说话.md`
6. `chapter_06_把项目放进简历.md`

读法建议：先读故事，再看每章末尾的“这一章你要记住”。如果某一章看不懂，先不用硬啃代码，回到网页上点一遍对应功能，会更容易连起来。
